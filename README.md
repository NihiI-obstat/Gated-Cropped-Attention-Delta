# GCAD Persona Steering

## Setup

```bash
pip install -r requirements.txt
export OPENAI_API_KEY=<your_api_key>
```

```bash
export MODEL_PATH_OR_ID=/path/to/Qwen2.5-7B-Instruct
```

## Contents

- `data/`: trait instructions, extraction questions, and evaluation questions.
- `eval_persona_extract/`: cached positive/negative extraction CSVs used to regenerate vectors without rerunning judged extraction.
- `vectors/original/`: original residual-stream persona vectors.
- `vectors/prompt_attn_delta/`: GCAD attention-value vectors.
- `vectors/prompt_attn_k_pos/`: GCAD key and demand calibration vectors.
- `scripts/generate_vec.sh`: original vector extraction.
- `scripts/generate_vec_prompt_attn_delta.sh`: GCAD attention-delta vector extraction.
- `scripts/generate_vec_prompt_attn_k.py`: GCAD key/demand vector extraction.
- `scripts/eval_steering_multiturn_original.sh`: original multi-turn evaluation.
- `gcad/run.sh`: GCAD multi-turn evaluation.

## Reproduce Main Runs

Original residual-stream steering:

```bash
bash scripts/eval_steering_multiturn_original.sh 0,1
```

GCAD:

```bash
bash gcad/run.sh 0,1
```

The scripts write outputs under `results/`.

## Regenerate Vectors

Original vectors:

```bash
bash scripts/generate_vec.sh 0,1
```

GCAD attention vectors:

```bash
CUDA_VISIBLE_DEVICES=0,1 bash scripts/generate_vec_prompt_attn_delta.sh
```

GCAD key/demand vectors:

```bash
CUDA_VISIBLE_DEVICES=0,1 python scripts/generate_vec_prompt_attn_k.py
```
